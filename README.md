# Binary conversions
**Tiny C program for converting to decimal, binary, octal and hexadecimal**

### Usage

#### Decimal
Syntax:
```bash
conversor.out d <decimal integer>  # Note that 'd' can be any word starting with 'd', like 'dec' or 'decimal'
```

Example:
```console
$ ./conversor.out decimal 420
Dec: 420
Bin: 110100100
Oct: 644
Hex: 1A4
```
